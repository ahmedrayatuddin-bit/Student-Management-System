# Student Management System  
(Java Swing + MySQL using JDBC)

## Project Overview
This project is a **Student Management System** developed using **Java Swing** for the graphical user interface and **MySQL** for the database.  
The application uses **JDBC** to connect Java with the MySQL database and supports **role-based access** for different users.

This project was developed as part of a **Database Systems course**.

---

## Technologies Used
- Java (Swing)
- MySQL
- JDBC
- NetBeans IDE

---

## Features
- Student Information Management
- Teacher Information Management
- Student Fee Management
- Leave Management System
- Role-based Login System
- Graphical User Interface (GUI)

---

## CRUD Operations
The system supports **complete CRUD operations**:
- **Create**: Add new student, teacher, and related records
- **Read**: View existing records
- **Update**: Modify existing information
- **Delete**: Remove records from the system

---

## User Roles
The system includes **multiple user roles**:
- **Admin**  
  - Full access to all modules and records
- **User (Student / Teacher)**  
  - Limited access based on permissions

## How to Run the Project
1. Install JDK
2. Install MySQL Server
3. Open **NetBeans IDE**
4. Open the project folder:  
`Student Management System`
5. Add the MySQL JDBC connector from the `mysql connector` folder
6. Update database connection credentials if needed
7. Run the project
